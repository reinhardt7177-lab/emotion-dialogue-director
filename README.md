# 감정 대화 감독관

초등학교 3~6학년 학생이 학교생활 속 갈등과 학교폭력 위험 신호를 살피고, 감정조절·공감·도움 요청을 연습하는 국어·생활·인성교육 웹 앱입니다.

## 주요 학습관

- `감정 대화 감독관` — 말과 표현을 고르고 선택에 따른 관계 변화 살펴보기
- `마음 멈춤 코치` — 감정이 커질 때 멈추고 안전한 다음 행동 고르기
- `관계 신호 탐정` — 갈등과 학교폭력 위험 신호를 구별하고 도움 요청하기
- `용기 있는 옆 친구` — 목격자의 안전한 방어 행동과 공동체 책임 연습하기

총 14개 학교생활 장면을 제공하며, 장면마다 감정 포착 → 말과 행동 선택 → 결과 팝업 → 관점 바꾸기 → 실천 문장 → 결말 비교 흐름으로 학습합니다.

## 실행

별도의 설치나 계정 없이 `index.html`을 열면 실행됩니다. 모든 그림은 `assets/` 폴더에 포함되어 있습니다.

## 교육 설계

- 대상: 초등학교 3~6학년
- 활용: 국어, 인성교육, 창의적 체험활동, 학교폭력 예방교육
- 핵심 역량: 감정 인식, 자기조절, 공감, 존중, 갈등 해결, 도움 요청, 방어 행동
- 참고 방향: 어울림 프로그램의 공감·의사소통·감정조절·갈등해결·학교폭력 인식 및 대처 요소

## 화면 구성

- 학생 친화적인 둥근 제목 글꼴과 큰 본문 글씨
- 장면과 결말을 크게 보여 주는 16:9 이미지
- 선택 결과를 즉시 비교하는 중앙 팝업
- `머리에 쏙! 기억 문장`과 핵심 낱말 카드
- 학생이 혼자 읽고 진행할 수 있는 단계별 안내와 안전 피드백
